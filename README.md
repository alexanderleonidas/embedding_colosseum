# Embedding Strategies for High-Dimensional Imaging in Quantum Machine Learning Pipelines

To start, download [uv](https://docs.astral.sh/uv/getting-started/installation/) and execute `uv sync` in your terminal.

To add packages to the project: `uv add PACKAGE_NAME`
To remove packages to the project: `uv remove PACKAGE_NAME`

Those commands will install the the packages in your local environment and add them to the project config or remove them respectively.
